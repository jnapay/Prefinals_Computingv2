# Generative AI Website

## Overview
This website explains Generative AI (GenAI) in a simple and interactive way. It covers:

- What Generative AI is
- The process behind it
- Examples of tools
- Strengths and weaknesses
- FAQs
- Ethical use of AI
- Analogy article on AI in development work

The website includes multiple pages and interactive elements using HTML, CSS, and optional JavaScript.

---

## Pages

| Page            | Description                                             |
|------           |-------------                                            |
| `index.html`    | Home / Landing page introducing Generative AI           |
| `faq.html`      | Frequently Asked Questions section                      |
| `ethics.html`   | Ethical considerations of using AI                      |
| `article.html`  | Analogy or detailed article about AI in development work |

---

## How to Run

1. Clone or download the repository.  
2. Open `index.html` in any web browser.  
3. No server setup is required; works locally.  

library used - Bootstrap
